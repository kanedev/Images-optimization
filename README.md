## 🚀 Install

```sh
$ git clone https://github.com/kanedev/images-optimization
$ cd createimage
$ yarn 
$ yarn start 
```



## :sparkles: Features

⚡️  To resize images :

```sh
npm run resize
```

⚡️ To compress images :

```sh
npm run compress
```

⚡️ To transform text to image : 

```sh
npm run txt2img
```
