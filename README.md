## Install

```sh
npm install
```

## Usage

>> To start the app
```sh
npm run start
```

>> To resize images :
```sh
npm run resize
```

>> To compress images : 
```sh
npm run compress
```

>> To transform text to image : 
```sh
npm run txt2img
```
